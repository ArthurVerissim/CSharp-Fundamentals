# Space Expedition

Um script simples de console pra testar estruturas de decisão e operadores lógicos. 

## Tech
* C#

## O que o código faz
Avalia condições de uma expedição espacial usando variáveis booleanas:
* Usa `&&` para checar se o planeta é habitável (ar + gravidade).
* Usa `||` para ver se é viável expandir (gravidade ou recursos).
* Usa `<` para verificar se ainda cabe tripulação na nave.

## Notas / Desafios
Foi meio chato ter que ficar inventando as lógicas e tentando forçar um contexto espacial só pra fazer os operadores booleanos se encaixarem e fazerem sentido.
