# 🚀 Agência de Viagens Galáctica

Projeto prático em **C#** focado nos fundamentos de manipulação de variáveis e conversão de tipos de dados (implícita, explícita e métodos da classe `Convert`).

## ⚙️ Como executar

Certifique-se de ter o [.NET SDK](https://dotnet.microsoft.com/download) instalado. No terminal, execute:

```bash
# Entre na pasta do projeto
cd 01-ManipulacaoDeDados

# Rode a aplicação
dotnet run
